//Cómo ejecutar el programa//
El programa esta basado en bucles for y un condicional if acompañado de una libreria llamada intertools que nos ayudara a manipular los iteradores
de manera mas facil.
//Ejemplos de salida//
Contraseña 'alp' encontrada.
Intentos totales: 1004
//¿qué pasa si la contraseña tiene 8+ caracteres y usa mayúsculas, números y símbolos?//
En este caso el programa esta diseñado para ser capaz de decifrar contraseñas de mas caracteres ya que se incluyo un for
donde podemos definir la longitud de los caracteres que posiblemente tenga la contraseña
